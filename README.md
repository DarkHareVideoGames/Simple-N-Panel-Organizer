  <h1>Simple N-Panel Organizer</h1>
  <img src="images/screenshot.png" alt="Simple N-Panel Organizer UI" class="screenshot">
  <p>Organize, reorder, and resize your Blender N-panel tabs with a simple, persistent UI. Save your preferred layout and restore it automatically every session.</p>
  <a href="MANUAL.md" class="manual-link">Read the Manual</a>
  <h2>Features</h2>
  <ul>
    <li>Drag-and-drop tab reordering</li>
    <li>Resize tabs for better visibility</li>
    <li>Automatic saving and restoring of tab order</li>
    <li>Easy-to-use interface in the N-panel</li>
  </ul>
  <h2>Installation</h2>
  <ol>
    <li>Download <code>reorder_npanel_tabs.py</code> from this repository.</li>
    <li>In Blender, go to <strong>Edit &gt; Preferences &gt; Add-ons</strong>.</li>
    <li>Click <strong>Install</strong> and select the downloaded file.</li>
    <li>Enable the addon in the list.</li>
  </ol>
