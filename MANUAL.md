# Simple N-Panel Organizer Manual

## Introduction
Simple N-Panel Organizer is a Blender addon that lets you easily reorder, organize, and resize your N-Panel tabs. Tab order is persistent and automatically saved when you apply changes.

## Features
- Drag and reorder N-Panel tab headers
- Move tabs up/down with buttons
- Apply tab order (automatically saves)
- Tab order persists across Blender sessions and workspace switches
- Resizable tab list UI

## Installation
1. Download `reorder_npanel_tabs.py` from the repository.
2. Open Blender and go to `Edit > Preferences > Add-ons`.
3. Click `Install...`, select `reorder_npanel_tabs.py`, and click `Install Add-on`.
4. Enable "Simple N-Panel Organizer" in the Add-ons list.

## Usage
1. Open the Sidebar (N-Panel) in the 3D View (`N` key).
2. Find the "Simple N-Panel Organizer" panel.
3. Use the list to reorder tabs:
   - Select a tab and use the Up/Down buttons to move it.
   - Click "Apply Tab Order" to apply and save the new order.
4. The tab order will persist automatically.

## Troubleshooting
- If the addon does not appear, ensure it is enabled in Preferences.
- If you see errors, check for duplicate or missing imports in the addon file.
- For permission errors, make sure Blender is closed when updating the addon file.

## FAQ
**Q:** Do I need to manually save the tab order?
**A:** No, saving is automatic when you apply the tab order.

**Q:** Will my tab order persist after restarting Blender?
**A:** Yes, the order is saved and restored automatically.
