## Installation
1. Download `reorder_npanel_tabs.py` from the repository.
2. In Blender, go to **Edit > Preferences > Add-ons**.
3. Click **Install** and select the downloaded file.
4. Enable the addon in the list.

---

## Usage
- Open the N-panel in Blender (press **N**).
- Use the Simple N-Panel Organizer UI to drag, reorder, and resize tabs.
- Click **Apply** to save your changes. The order will persist across sessions.

---

## Troubleshooting
- If the tab order does not persist, ensure Blender has permission to write files in your workspace.
- For UI issues, try disabling and re-enabling the addon.
